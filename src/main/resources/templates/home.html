Github: https://github.com/gagumareu/springboot_campboard

사용 기술

무중단 배포 버전: Spring boot 2.3.x jar 개발 버전: Spring boot 2.7.x war JPA Querydsl Spring security thymeleaf Ajax summernote editor Google OAuth2 AWS EC2 AWS RDS AWS S3 Bucket

개발 목표

  • 유저가 등록한 캠핑 장비를 리스트업
  • 리스트업한 장비들에서 선택하여 중고 거래 작성
  • list.html 한 페이지에서 카테고리 별로 리스트 출력 함으로써 최소한의 코드로 원하는 리스트 출력
  • 검색기능 '카테고리 내에서 원하는 키워드 검색가능'
  • Spring Security를 이용하여 권한이 주어진 회원만 게시물 등록, 수정 및 삭제
  • text edition 'summernote edition' 적용

개선 및 업데이트

  • 중고거래 등록된 캠핑 장비 수정 불가 처리
  • 회원 프로파일 이미지 수정
  • 회원 이름 클릭시 회원 정보 보이기
  • 유저가 등록한 캠핑 장비 검색 기능
  • 댓글 작성자 프로파일 이미지 보이기
  • 오픈 소스 map API 이용하여 캠핑 장소 모집
  • 게시물 및 장비 리스트 오름차순 내림차순 정렬
  • JWT 적용
  • SSL 인증
  • 첨부파일 업로드 로딩 표현
  • 모바일 버전 반응형
  • css link 처리
  • 마이 기어 클릭시 상제 정보 모달창 띄우기
  • 중고거래 리스트 썸네일 이미지 띄우기
